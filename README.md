## Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2: GitHub
1. Push this folder to a GitHub repo
2. Go to vercel.com
3. Import the repo
4. Deploy

### Option 3: Drag & Drop
1. Go to vercel.com/drop
2. Drag this folder
3. Done!

## Files
- `index.html` - Main app
- `buch-*.md` - Book chapters
- `vercel.json` - Vercel config
